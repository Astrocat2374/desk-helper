Instructions

Desk Area Width:

One side of the total desk area. The program will prioritize using up this entire side to shorten the overall length.

Desk Area Length:

The side meant to be compressed. The program will try to minimize this side as much as possible.

Minimum Distance Between Desks:

The minimum distance required between each desk.

Number of Desks:

The number of desks you are trying to fit into the space. The program will find the most efficient layout based on the width and will let you know if the length is too short for all the desks.

Note:

The app works assuming all measurements are from the center of desks. This means that the classroom length and width should be appropriately shortened to allow for the edges of desks to be away from walls. A feature will be implemented soon to make this process easier.
